オフセット、クエヴォの発表を受けてミーゴスの再結集について言及

POST LINE
4月27日、クエヴォ（Quavo）がインスタグラムで投稿したスローバック画像がミーゴスの今後について新たな憶測を呼び起こした。テイクオフ（Takeoff）との写真とともに「Warriors Never fold」というキャプションを添え、その後「Jobs Not Finished. TAKEOFF ALBUM. UNC N PHEW 2. LAST ????? ALBUM. REAL MIGO BLOOD RUN IN MY VIENS!!! AINT NO NEW CHAPTER JUST THE NEXT ONE!!!」という長いメッセージを投稿した。このトーンは終わりではなく継続性を示唆していた。 数時間以内に、オフセット（Offset）は2021年のトリオの写真を投稿し、「On dat!!!」と書き込んだ。この投稿により、グループが何らかの形で再び協力し合う可能性についての憶測が再燃した。 ミーゴスの最後のフルプロジェクト『Culture III』は2021年にリリースされ、その後の変化の時期が来る前のピークを示していた。その直後、クエヴォとテイクオフは『Only Built for Infinity Links』をリリースし、デュオとしてのパートナーシップにフォーカスをシフトさせた。しかし、テイクオフがヒューストンのボウリング場外で銃撃され殺害されたことにより、その勢いは突然止まった。この喪失はグループの軌跡を変え、その将来を不確実にした。 以来、公開の場での出演やコラボレーションは散発的であり、ファンはそれぞれの動きについて意味を探ってきた。 オフセットは「7PM in Brooklyn」（ゲストはカルメロ・アンソニー）への出演時に、自分たちの関係について直接言及した。「インターネットが昔のようなことをしようとするが、俺たちにはそんなことじゃない」と述べた。「俺たちはお互いに声をかけ合う必要があるんだ。結局のところ、それは家族だ。時には家族と意見が合わないこともあるかもしれない。その時点では、一方が他方と遊ぶつもりはないかもしれない。」別の会話では『Baby, This Is Keke Palmer』でオフセットはサウィーティに関する憶測に言及し、彼女が以前の対立に何らかの役割を果たしたという考えを否定した。
