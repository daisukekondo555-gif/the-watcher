Xbox Game Pass Ultimateが値下げ。新CEOアシャ・シャルマが価格戦略を転換

POST LINE
新しいXbox CEOのアシャ・シャルマ（Asha Sharma）は、フィル・スペンサーの退任とサラ・ボンドの退社に続いて、同社を新しい方向へ推し進めている。シャルマはXbox Game Pass Ultimateの値下げを発表した。シャルマは値下げの理由をGame Pass Ultimateが「多くのプレイヤーにとって高すぎる」ことだと説明しているが、同時に将来のCall of Dutyゲームが同サービスの発売日に利用可能にならなくなることも影響している。現在のCall of Dutyゲームはサービス内に残るが、将来のゲームはシャルマによると「約1年後」の発売翌年のホリデーシーズンに参入するという。 本日より、Game Pass Ultimateは月額29.99ドルから22.99ドルに値下げされた。PC Game Passも月額16.49ドルから13.99ドルに値下げされている。シャルマがX（旧Twitter）での投稿で述べたところによると、現在のCall of Dutyタイトルはプレイヤーにとって重要な事柄にGame Passをより良くマッチさせるために、引き続き学習と進化を続けるという。 この発表は、シャルマがサービスが高すぎることについて言及した内部メモがリークされたことに続くものである。Game Pass Ultimateの価格が引き下げられたことで、残された課題はXboxが現在の戦略を見直し、現在苦戦しているハードウェアの購入を促すための独占的なファーストパーティゲームに戻すかどうかであり、シャルマがそのような方針に戻すかどうか注視する必要がある。 ゲーマーたちはGame Pass Ultimateの値下げのニュースに好意的に反応している。
